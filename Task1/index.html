Basic

  • git config --global user.name blabla - задавання ім`я
  • git config --global user.email blabal@bla - задавання e-mail
  • git init - ініціалізація порожнього локального git-репозиторія
  • git status - перевіряння стану
  • git add - додавання файлів
  • git commit - покладення змін в локальний репозиторій
  • git log - перегляд історії комітів
  • git checkout - переключення гілки
  • git branch - показ лише локальних гілок
  • git push - покладення змін з локального до віддаленого репозиторію
  • git pull - забрання змін з віддаленого репозиторію по поточній гілці
  • git merge - злиття окремих гілок в єдину
  • git clone - клонування репозиторія
  • .gitignore - приховування файлів

Advanced

  • git revert - видалення комітів з гілки
  • git reset - скасування змін
  • git rebase - відтворення усіх змін, що були в комітах однієї гілки, на іншій гілці, перебазовування
  • git cherry-pick - додавання деяких комітів однієї гілки до іншої гілки
  • git stash - тимчасове приховування змін
